台灣節慶活動查詢
---------------------------------------------------------------
功能:
- 搜尋關鍵字活動查詢
- 依照地區分類搜尋
- 依照活動截止時間排序
- 點擊活動可察看詳細活動說明

Technologies
--------------------------------
- Front-End Fundamental
  - HTML / CSS / JavaScript
  - SCSS
  - RWD
- Front-End Frameworks
  - React
  - React Context API
  - React Router
- Third-party API
